# A Thought in the Mind of God -- Book Build System

Self-contained build pipeline that turns the book's markdown into web HTML, web PDF,
print interiors (7x10 color, 7x10 B&W, 8.5x11, 6x9), and EPUB.

See `docs/` for full documentation. Quickstart once implemented:

    engine/build.sh 1st-edition all

This repo is built per the spec/plan in PristineGrace/docs/superpowers/.
Phase 1 = byte-identical relocation + config skeleton. (in progress)
